== Description ==

Image Caption Hover Effects is a <strong>Multi Purpose</strong> Plugin to displays Images and Captions with 70+ CSS3 Hover Animations. Responsive Grid for Images along with Pop Up Support. It Includes following incredible features.
<h3><a href="http://demos.webcodingplace.com/image-caption-hover-effects-demo/">View Live Demo</a></h3>
<h3>Features</h3>
<ul>
<li>70+ Modern Hover Animations</li>
<li>Responsive Grid (1 to 12 Columns)</li>
<li>Responsive Light Box</li>
<li>Drag Drop to Sort Columns</li>
<li>Import and Export Capability</li>
<li>Categories Support</li>
<li>Dummy Content Included</li>
</ul>

== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Image Caption Hover Effects' and install it
3. Go to IHC Effects > Add New, and insert columns
4. Upload images and fill some additional informations. (Not Necessary)
5. Now paste shortcode of that effect and Enjoy...
